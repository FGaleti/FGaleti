<h2 id="areas-of-expertise">Areas of Expertise</h2>

**Observability &amp; Monitoring**
Deployment and maintenance of monitoring stacks (Zabbix, Grafana), building dashboards and alerts
aimed at reducing detection and resolution time for incidents in critical environments.

**Operational Automation**
Development of scripts in Python, Shell Script, and Windows PowerShell to automate infrastructure
tasks, integrate with REST APIs, and orchestrate jobs via RunDeck.

**Server &amp; Network Administration**
Administration of Linux and Windows servers, remote access and management via SSH and RDP, along
with network administration and support for production environments.

**Virtualization &amp; Backup**
Operation of virtualized environments on VMware and Hyper-V, managing backup and recovery routines
with Veeam.

**Data**
Querying and maintaining relational databases (SQL, PostgreSQL, SQLite), applying data analysis to
support operational decisions and identify patterns in monitored environments.

**Security &amp; Compliance**
Applying information security and data protection best practices aligned with LGPD across
infrastructure processes.
